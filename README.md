# 📊 Student Placement and Internship Analytics Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard developed to analyze student placement and internship trends using survey data. The dashboard transforms raw data into meaningful insights through interactive visualizations.

---

## 🎯 Objectives

- Analyze student placement trends
- Monitor internship participation
- Compare department-wise participation
- Track placement attempts
- Generate actionable insights

---

## 📂 Dataset

- Google Form Survey Responses
- 50 Student Records
- 36 Colleges
- Multiple Departments

> **Note:** The dataset shared in this repository has been anonymized to protect personal information.

---

## 🛠 Tools Used

- Google Forms
- Microsoft Excel
- Microsoft Power BI

---

## 📈 Dashboard Features

- KPI Cards
- Interactive Slicers
- Department-wise Analysis
- Internship Status Distribution
- Placement Offer Analysis
- Placement Attempts Distribution
- Academic Year Analysis
- Key Insights Panel

---

## 💡 Key Insights

- 50 student responses analyzed
- 36 colleges covered
- CSE recorded the highest participation
- 76% of students expressed interest in internships
- 30% of students received placement offers
- Average placement attempts: **3.76**
